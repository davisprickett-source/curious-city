# Anchorage Dark History - Archival Image Sources Guide

## Direct Links to Authentic Historical Images

This guide provides direct links to archival sources for authentic Anchorage historical images.

---

## 1. Alaska Digital Archives (VILDA)
**URL:** https://vilda.alaska.edu/

### Search Queries:
- "Anchorage Fourth Avenue 1970s"
- "Trans-Alaska Pipeline Anchorage"
- "Good Friday Earthquake 1964"
- "Anchorage downtown 1980"
- "Ship Creek Anchorage"
- "Eklutna Alaska"
- "Sterling Alaska"

### How to Download:
1. Search for the image
2. Click on the image
3. Click "Download" button (usually top right)
4. Select high-resolution option
5. Save with descriptive filename

## Copyright and Attribution Guidelines

### Public Domain (No Permission Needed):
- All USGS images
- All NOAA images
- Library of Congress (most)
- National Archives (most)
- Federal government photos

### Permission Required:
- Museum collections (Anchorage Museum)
- Private hotel archives
- Newspaper archives
- State archives (some)

### Proper Attribution Format:
```
Government: "U.S. Geological Survey (Public Domain)"
Museum: "Anchorage Museum, [Collection Name]"
Library: "Library of Congress, [Collection ID]"
```

## Direct Download Commands

Once you have direct image URLs, use:

```bash
# For USGS/NOAA (Public Domain)
curl -L "https://[direct-image-url]" -o filename.jpg

# For Wikimedia Commons
curl -L "https://commons.wikimedia.org/wiki/Special:FilePath/[filename]" -o local-filename.jpg

# For Library of Congress
curl -L "https://tile.loc.gov/storage-services/[path]" -o filename.jpg
```
