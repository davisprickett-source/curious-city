## Image Download Instructions

### Using Google Maps:
1. Search for the address
2. Click "Photos"
3. Right-click and "Save Image As"
4. Verify it's not copyrighted (look for user-submitted photos)

### Using Yelp:
1. Search business name
2. Browse photos section
3. Look for user-submitted photos (not stock photos)
4. Download with proper attribution

### Using Instagram:
1. Search hashtag or handle
2. Find posts with good lighting and composition
3. Use an Instagram downloader tool
4. Always credit the photographer

### File Size Requirements:
- Images should be at least 1200px wide
- Under 500KB after optimization
- JPG format preferred
- Filenames should be descriptive and kebab-case
