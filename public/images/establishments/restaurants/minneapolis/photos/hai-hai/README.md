## Image Quality Guidelines

- **Minimum Resolution:** 1200px width
- **Format:** JPG (for photos)
- **Aspect Ratio:** 3:2 or 4:3 preferred
- **File Size:** 200KB - 2MB optimal for web
- **Color Profile:** sRGB for web display

## Attribution Best Practices

Even when not required, consider:
